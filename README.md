# Pathogen Forecasting Model — Web Dashboard

Interactive visualization of near-real-time sewage/pathogen concentration forecasts along the San Diego / Tijuana coastline, produced by the [Feddersen Lab](https://feddersen.ucsd.edu) at Scripps Institution of Oceanography, UCSD.

Live site: **[pfmweb.ucsd.edu](https://pfmweb.ucsd.edu)**

---

## Overview

This dashboard displays output from a numerical ocean circulation model that tracks sewage tracer (dye) concentration in the coastal ocean near the Tijuana River outfall. Forecasts are generated daily and cover a ~5-day window at hourly resolution.

### Monitoring sites

| Site | Risk thresholds |
|------|----------------|
| Playas de Tijuana | Low < 0.001% < Medium < 0.1% < High |
| Imperial Beach Pier | " |

> **`data/` is not committed.** It is regenerated locally from `web_data_latest.nc` whenever the forecast updates.

## How it works

### Data pipeline (`preprocess.py`)

The model output is stored in a NetCDF4 file (`web_data_latest.nc`, ~1 GB) with the following key variables:

| Variable | Shape | Description |
|----------|-------|-------------|
| `map_l10_dye_tot` | (121, 1141, 486) | log₁₀ dye concentration on a curvilinear grid |
| `shoreline_risk` | (121, 1060) | Risk category (0/1/2) at 1060 shoreline points |
| `sites_dye_tot` | (121, 4) | Dye concentration at the 4 monitoring sites |
| `time` | (121,) | Days since 1999-01-01 |

The model uses a **curvilinear ROMS grid** (rows/columns are not aligned with lat/lon). `preprocess.py` re-projects the data onto a regular lat/lon raster using a KD-tree nearest-neighbour lookup, then applies a colour ramp and saves each of the 121 time steps as a transparent RGBA PNG (~50 KB each). It also writes compact JSON files for the site time series and shoreline risk.

### Dashboard (`index.html`)

A self-contained static page with no build step or framework dependency:

- **Map** — [Leaflet](https://leafletjs.com) with an Esri satellite basemap. Pre-rendered PNG frames are swapped in as an image overlay as the user scrubs through time. Shoreline points are rendered as coloured `circleMarker`s.
- **Colormap** — transparent → yellow → orange → dark purple, log-scaled. Matches the risk thresholds: yellow visible at medium risk (> 0.001%), orange at high risk (> 0.1%), dark purple above ~5%.
- **Time series chart** — [Chart.js](https://www.chartjs.org) line plot of log₁₀ concentration at each monitoring site with threshold reference lines.
- **Playback** — play/pause button animates through all 121 frames at ~7 fps; adjacent frames are preloaded in the background.

## Setup

### Requirements

Uses the `bathy_compare-env` conda environment (Python 3.14):

```
h5py
numpy
scipy
Pillow
```

### Generate web assets

```bash
cd pfm-website/
# Place web_data_latest.nc in this directory, then:
python preprocess.py
```

Outputs to `data/` (~12 MB total). Takes about 2 minutes on a laptop.

### Preview locally

```bash
python -m http.server 8765
# open http://localhost:8765
```

### Update forecast

When a new `web_data_latest.nc` is available, regenerate and commit:

```bash
python preprocess.py
# deploy data/ to the web server
```
